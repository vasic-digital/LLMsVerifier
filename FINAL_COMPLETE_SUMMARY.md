## üöÄ DEPLOYMENT INSTRUCTIONS

### Quick Start (5 minutes)
```bash
# Clone repository
git clone https://github.com/llm-verifier/llm-verifier.git
cd llm-verifier

# Run verification (all 395 models)
./run_challenges

# Check results
ls -la ~/Downloads/LLM-Verifier-Reports/
```

### Website Deployment
```bash
# Build for production
cd llm-verifier/web
npm install
ng build --configuration production

# Serve locally
ng serve --host 0.0.0.0 --port 4200

# Deploy to hosting (example: Netlify)
ng build --configuration production
netlify deploy --dir=dist/llm-verifier-web --prod
```

### Docker Deployment
```bash
# Build Docker image
docker build -t llm-verifier:latest .

# Run with default config
docker run -p 8080:8080 llm-verifier:latest

# Run with custom config
docker run -p 8080:8080 -v $(pwd)/config.yaml:/config llm-verifier:latest --config /config/config.yaml
```

### Kubernetes Deployment
```bash
# Apply deployment
kubectl apply -f k8s-deployment.yaml

# Check status
kubectl rollout status deployment/llm-verifier

# Scale based on load
kubectl scale deployment/llm-verifier --replicas=5
```
