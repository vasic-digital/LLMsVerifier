Getting Started in 3 Steps

1

Install LLM Verifier

go install github.com/your-org/llm-verifier@latest

Or download pre-built binaries for your platform

2

Configure API Keys

export OPENAI_API_KEY="sk-your-key"
export ANTHROPIC_API_KEY="sk-ant-your-key"

Set up your AI provider API keys securely

3

Start Verifying

llm-verifier verify --provider openai --model gpt-4o
llm-verifier export-config opencode --output config.json
